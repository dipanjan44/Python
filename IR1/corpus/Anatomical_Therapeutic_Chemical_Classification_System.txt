<h1 class="firstHeading" id="firstHeading" lang="en">Anatomical Therapeutic Chemical Classification System</h1> <div class="mw-body-content" id="bodyContent">
<div class="noprint" id="siteSub">From Wikipedia, the free encyclopedia</div> <div id="contentSub"></div>
<div id="jump-to-nav"></div> <a class="mw-jump-link" href="#mw-head">Jump to navigation</a>
<a class="mw-jump-link" href="#p-search">Jump to search</a>
<div class="mw-content-ltr" dir="ltr" id="mw-content-text" lang="en"><div class="mw-parser-output"><p>The <b>Anatomical Therapeutic Chemical (ATC) Classification System</b> is used for the <a href="/wiki/Drug_class" title="Drug class">classification</a> of <a href="/wiki/Medication" title="Medication">active ingredients</a> of <a href="/wiki/Drug" title="Drug">drugs</a> according to the <a href="/wiki/Organ_(anatomy)" title="Organ (anatomy)">organ</a> or <a href="/wiki/System" title="System">system</a> on which they act and their <a class="mw-redirect" href="/wiki/Therapeutic" title="Therapeutic">therapeutic</a>, <a class="mw-redirect" href="/wiki/Pharmacological" title="Pharmacological">pharmacological</a> and chemical properties. It is controlled by the <a href="/wiki/World_Health_Organization" title="World Health Organization">World Health Organization</a> Collaborating Centre for Drug Statistics Methodology (WHOCC), and was first published in 1976.<sup class="reference" id="cite_ref-history_1-0"><a href="#cite_note-history-1">[1]</a></sup>
</p><p>This <a href="/wiki/Pharmaceutical_code" title="Pharmaceutical code">pharmaceutical coding system</a> divides drugs into different groups according to the organ or system on which they act, their <a href="/wiki/Medicine" title="Medicine">therapeutic</a> intent or nature, and the drug's <a href="/wiki/Chemical_classification" title="Chemical classification">chemical characteristics</a>. Different brands share the same code if they have the same active substance and indications. Each bottom-level ATC code stands for a pharmaceutically used substance, or a combination of substances, in a single indication (or use). This means that one drug can have more than one code, for example <a class="mw-redirect" href="/wiki/Acetylsalicylic_acid" title="Acetylsalicylic acid">acetylsalicylic acid</a> (aspirin) has <a href="/wiki/ATC_code_A01" title="ATC code A01">A01AD05</a> (<span title="www.whocc.no"><a class="external text" href="https://www.whocc.no/atc_ddd_index/?code=A01AD05" rel="nofollow">WHO</a></span>) as a drug for local <a href="/wiki/Mouth" title="Mouth">oral</a> treatment, <a href="/wiki/ATC_code_B01" title="ATC code B01">B01AC06</a> (<span title="www.whocc.no"><a class="external text" href="https://www.whocc.no/atc_ddd_index/?code=B01AC06" rel="nofollow">WHO</a></span>) as a <a class="mw-redirect" href="/wiki/Platelet_inhibitor" title="Platelet inhibitor">platelet inhibitor</a>, and <a href="/wiki/ATC_code_N02" title="ATC code N02">N02BA01</a> (<span title="www.whocc.no"><a class="external text" href="https://www.whocc.no/atc_ddd_index/?code=N02BA01" rel="nofollow">WHO</a></span>) as an <a href="/wiki/Analgesic" title="Analgesic">analgesic</a> and <a href="/wiki/Antipyretic" title="Antipyretic">antipyretic</a>); as well as one code can represent more than one active ingredient, for example <a href="/wiki/ATC_code_C09" title="ATC code C09">C09BB04</a> (<span title="www.whocc.no"><a class="external text" href="https://www.whocc.no/atc_ddd_index/?code=C09BB04" rel="nofollow">WHO</a></span>) is the combination of perindopril with amlodipine, two active ingredients that have their own codes (<a href="/wiki/ATC_code_C09" title="ATC code C09">C09AA04</a> (<span title="www.whocc.no"><a class="external text" href="https://www.whocc.no/atc_ddd_index/?code=C09AA04" rel="nofollow">WHO</a></span>) and <a href="/wiki/ATC_code_C08" title="ATC code C08">C08CA01</a> (<span title="www.whocc.no"><a class="external text" href="https://www.whocc.no/atc_ddd_index/?code=C08CA01" rel="nofollow">WHO</a></span>) respectively) when prescribed alone.
</p><p>The ATC classification system is a strict hierarchy, meaning that each code necessarily has one and only one parent code, except for the 14 codes at the topmost level which have no parents. The codes are semantic identifiers, meaning they depict in themselves the complete lineage of parenthood.
</p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The ATC system is based on the earlier Anatomical Classification System, which is intended as a tool for the pharmaceutical industry to classify pharmaceutical products (as opposed to their active ingredients).<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> This system, confusingly also called ATC, was initiated in 1971 by the <a class="new" href="/w/index.php?title=European_Pharmaceutical_Market_Research_Association&amp;action=edit&amp;redlink=1" title="European Pharmaceutical Market Research Association (page does not exist)">European Pharmaceutical Market Research Association</a> (EphMRA) and is being maintained by the EphMRA and the <a class="new" href="/w/index.php?title=Pharmaceutical_Business_Intelligence_and_Research_Group&amp;action=edit&amp;redlink=1" title="Pharmaceutical Business Intelligence and Research Group (page does not exist)">Pharmaceutical Business Intelligence and Research Group</a> (PBIRG). Its codes are organised into four levels.<sup class="reference" id="cite_ref-3"><a href="#cite_note-3">[3]</a></sup> The WHO's system, having five levels, is an extension and modification of the EphMRA's. It was first published in 1976.<sup class="reference" id="cite_ref-history_1-1"><a href="#cite_note-history-1">[1]</a></sup>
</p>
<h2><span class="mw-headline" id="Classification">Classification</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=2" title="Edit section: Classification">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>In this system, drugs are classified into groups at five different levels:<sup class="reference" id="cite_ref-4"><a href="#cite_note-4">[4]</a></sup>

<h3><span class="mw-headline" id="Second_level">Second level</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=4" title="Edit section: Second level">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The second level of the code indicates the therapeutic subgroup and consists of two digits.<sup class="reference" id="cite_ref-6"><a href="#cite_note-6">[6]</a></sup>
</p><p><i>Example</i>: <a href="/wiki/ATC_code_C03" title="ATC code C03">C<b>03</b></a> <a href="/wiki/Diuretic" title="Diuretic">Diuretics</a>
</p>
<h3><span class="mw-headline" id="Third_level">Third level</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=5" title="Edit section: Third level">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The third level of the code indicates the therapeutic/pharmacological subgroup and consists of one letter.
</p><p><i>Example</i>: C03<b>C</b> High-ceiling diuretics
</p>
<h3><span class="mw-headline" id="Fourth_level">Fourth level</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=6" title="Edit section: Fourth level">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The fourth level of the code indicates the chemical/therapeutic/pharmacological subgroup and consists of one letter.
</p><p><i>Example</i>: C03C<b>A</b> <a href="/wiki/Sulfonamide_(medicine)" title="Sulfonamide (medicine)">Sulfonamides</a>
</p>
<h3><span class="mw-headline" id="Fifth_level">Fifth level</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=7" title="Edit section: Fifth level">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The fifth level of the code indicates the chemical substance and consists of two digits.
</p><p><i>Example</i>: C03CA<b>01</b> <a href="/wiki/Furosemide" title="Furosemide">furosemide</a>
</p>
<h3><span class="mw-headline" id="ATCvet">ATCvet</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=8" title="Edit section: ATCvet">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>The <i>Anatomical Therapeutic Chemical Classification System for veterinary medicinal products</i> (ATCvet) is used to classify <a class="mw-redirect" href="/wiki/Veterinary" title="Veterinary">veterinary</a> drugs. ATCvet codes can be created by placing the letter Q in front of the ATC code of most human medications. For example, furosemide for veterinary use has the code <b>Q</b>C03CA01.
</p><p>Some codes are used exclusively for veterinary drugs, such as <i><a href="/wiki/ATCvet_code_QI" title="ATCvet code QI">QI</a> Immunologicals</i>, <i><a href="/wiki/ATCvet_code_QJ51" title="ATCvet code QJ51">QJ51</a> Antibacterials for intramammary use</i> or <i>QN05AX90 <a href="/wiki/Amperozide" title="Amperozide">amperozide</a></i>.<sup class="reference" id="cite_ref-7"><a href="#cite_note-7">[7]</a></sup>
</p>
<h2><span class="mw-headline" id="Defined_daily_dose">Defined daily dose</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=9" title="Edit section: Defined daily dose">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="hatnote navigation-not-searchable" role="note">Main article: <a href="/wiki/Defined_daily_dose" title="Defined daily dose">Defined daily dose</a></div>
<p>The ATC system also includes defined daily doses (DDDs) for many drugs. This is a measurement of drug consumption based on the usual daily dose for a given drug. According to the definition, "[t]he DDD is the assumed average maintenance dose per day for a drug used for its main indication in adults."<sup class="reference" id="cite_ref-8"><a href="#cite_note-8">[8]</a></sup>
</p>
<h2><span class="mw-headline" id="Adaptations_and_updates">Adaptations and updates</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Anatomical_Therapeutic_Chemical_Classification_System&amp;action=edit&amp;section=10" title="Edit section: Adaptations and updates">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>National issues of the ATC classification, such as the <a href="/wiki/Germany" title="Germany">German</a> <i>Anatomisch-therapeutisch-chemische Klassifikation mit Tagesdosen</i>, may include additional codes and DDDs not present in the WHO version.<sup class="reference" id="cite_ref-9"><a href="#cite_note-9">[9]</a></sup>
</p><p>ATC follows guidelines<sup class="reference" id="cite_ref-10"><a href="#cite_note-10">[10]</a></sup> in creating new codes for newly approved drugs. In order to create a new ATC code, an application has to be sent to the WHO. New ATC codes are published twice annually.<sup class="reference" id="cite_ref-11"><a href="#cite_note-11">[11]</a></sup> A formal release of new ATC edition occurs once a year.
</p>

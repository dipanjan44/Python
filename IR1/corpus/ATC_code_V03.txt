<p><b>ATC code V03</b> <i>All other therapeutic products</i> is a therapeutic subgroup of the <a href="/wiki/Anatomical_Therapeutic_Chemical_Classification_System" title="Anatomical Therapeutic Chemical Classification System">Anatomical Therapeutic Chemical Classification System</a>, a system of alphanumeric codes developed by the <a href="/wiki/World_Health_Organization" title="World Health Organization">WHO</a> for the classification of <a class="mw-redirect" href="/wiki/Pharmaceutical_drug" title="Pharmaceutical drug">drugs</a> and other medical products. Subgroup V03 is part of the anatomical group <a href="/wiki/ATC_code_V" title="ATC code V">V <i>Various</i></a>.<sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>
</p><p>
Codes for <a class="mw-redirect" href="/wiki/Veterinary" title="Veterinary">veterinary</a> use (<a class="mw-redirect" href="/wiki/ATCvet_code" title="ATCvet code">ATCvet codes</a>) can be created by placing the letter Q in front of the human ATC code: for example, <b>QV03</b>.<sup class="reference" id="cite_ref-2"><a href="#cite_note-2">[2]</a></sup> ATCvet codes without corresponding human ATC codes are cited with the leading Q in the following list.</p><p class="mw-empty-elt"></p><p>National issues of the ATC classification may include additional codes not present in this list, which follows the WHO version.

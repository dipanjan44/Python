</p>

<h2><span class="mw-headline" id="Database">Database</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=1" title="Edit section: Database">edit</a><span class="mw-editsection-bracket">]</span></span></h2>

<h2><span class="mw-headline" id="Crowdsourcing">Crowdsourcing</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=2" title="Edit section: Crowdsourcing">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The ChemSpider database can be updated with user contributions including <a href="/wiki/Chemical_structure" title="Chemical structure">chemical structure</a> deposition, spectra deposition and user curation. This is a <a href="/wiki/Crowdsourcing" title="Crowdsourcing">crowdsourcing</a> approach to develop an online chemistry database. Crowdsourced based curation of the data has produced a <a href="/wiki/Dictionary" title="Dictionary">dictionary</a> of chemical names associated with chemical structures that has been used in <a class="mw-redirect" href="/wiki/Text-mining" title="Text-mining">text-mining</a> applications of the biomedical and chemical literature.<sup class="reference" id="cite_ref-16"><a href="#cite_note-16">[16]</a></sup>
</p><p>However, <a class="mw-redirect" href="/wiki/Database_rights" title="Database rights">database rights</a> are not waived and a <a class="mw-redirect" href="/wiki/Data_dump" title="Data dump">data dump</a> is not available; in fact, the FAQ even states that only limited downloads are allowed:<sup class="reference" id="cite_ref-17"><a href="#cite_note-17">[17]</a></sup> therefore the <a class="mw-redirect" href="/wiki/Right_to_fork" title="Right to fork">right to fork</a> is not guaranteed and the project can't be considered <a href="/wiki/Free_content" title="Free content">free</a>/<a href="/wiki/Open_data" title="Open data">open</a>.
</p>
<h2><span class="mw-headline" id="Searching">Searching</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=3" title="Edit section: Searching">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>A number of available search modules are provided:
</p>
<ul><li>The <i>standard search</i> allows <a href="/wiki/Information_retrieval" title="Information retrieval">querying</a> for <a href="/wiki/Systematic_name" title="Systematic name">systematic names</a>, <a href="/wiki/Trade_name" title="Trade name">trade names</a> and synonyms and registry numbers</li>
<li>The <i>advanced search</i> allows interactive searching by chemical structure, chemical substructure, using also molecular formula and <a class="mw-redirect" href="/wiki/Molecular_weight" title="Molecular weight">molecular weight</a> range, <a href="/wiki/Chemical_Abstracts_Service" title="Chemical Abstracts Service">CAS</a> numbers, suppliers, etc. The search can be used to widen or restrict already found results.</li>
<li>Structure searching on mobile devices can be done using free apps for <a href="/wiki/IOS" title="IOS">iOS</a> (iPhone/iPod/iPad)<sup class="reference" id="cite_ref-18"><a href="#cite_note-18">[18]</a></sup> and for the <a class="mw-redirect" href="/wiki/Android_operating_system" title="Android operating system">Android (operating system)</a>.<sup class="reference" id="cite_ref-19"><a href="#cite_note-19">[19]</a></sup></li></ul>
<h2><span class="mw-headline" id="Chemistry_document_mark-up">Chemistry document mark-up</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=4" title="Edit section: Chemistry document mark-up">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The ChemSpider database has been used in combination with <a href="/wiki/Text_mining" title="Text mining">text mining</a> as the basis of <a href="/wiki/Markup_language" title="Markup language">chemistry document markup</a>. ChemMantis,<sup class="reference" id="cite_ref-20"><a href="#cite_note-20">[20]</a></sup> the Chemistry Markup And Nomenclature Transformation Integrated System uses algorithms to identify and extract chemical names from documents and web pages and converts the chemical names to chemical structures using name-to-structure conversion algorithms and <a href="/wiki/Dictionary" title="Dictionary">dictionary</a> look-ups in the ChemSpider database. The result is an integrated system between chemistry documents and information look-up via ChemSpider into over 150 data sources.
</p>
<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=5" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>ChemSpider was acquired by the <a href="/wiki/Royal_Society_of_Chemistry" title="Royal Society of Chemistry">Royal Society of Chemistry</a> (RSC) in May, 2009.<sup class="reference" id="cite_ref-21"><a href="#cite_note-21">[21]</a></sup> Prior to the acquisition by RSC, ChemSpider was controlled by a private corporation, ChemZoo Inc. The system was first launched in March 2007 in a <a class="mw-redirect" href="/wiki/Beta_release" title="Beta release">beta release</a> form and transitioned to release in March 2008. ChemSpider has expanded the generic support of a chemistry database to include support of the <a href="/wiki/Wikipedia" title="Wikipedia">Wikipedia</a> chemical structure collection via their WiChempedia implementation.
</p>
<h2><span class="mw-headline" id="Services">Services</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=6" title="Edit section: Services">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>A number of services are made available online. These include the conversion of chemical names to <a href="/wiki/Chemical_structure" title="Chemical structure">chemical structures</a>, the generation of <a class="mw-redirect" href="/wiki/SMILES" title="SMILES">SMILES</a> and <a class="mw-redirect" href="/wiki/InChI" title="InChI">InChI</a> strings as well as the prediction of many physicochemical parameters and integration to a web service allowing <a class="mw-redirect" href="/wiki/NMR" title="NMR">NMR</a> prediction.
</p>
<h3><span class="mw-headline" id="SyntheticPages">SyntheticPages</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=7" title="Edit section: SyntheticPages">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>SyntheticPages is a free interactive database of <a class="mw-redirect" href="/wiki/Synthetic_chemistry" title="Synthetic chemistry">synthetic chemistry</a> procedures operated by the <a href="/wiki/Royal_Society_of_Chemistry" title="Royal Society of Chemistry">Royal Society of Chemistry</a>.<sup class="reference" id="cite_ref-22"><a href="#cite_note-22">[22]</a></sup> Users submit synthetic procedures which they have conducted themselves for publication on the site. These procedures may be original works, but they are more often based on literature reactions. <a class="mw-redirect" href="/wiki/Citations" title="Citations">Citations</a> to the original published procedure are made where appropriate.  They are checked by a scientific editor before posting. The pages do not undergo formal <a class="mw-redirect" href="/wiki/Peer-review" title="Peer-review">peer-review</a> like a <a href="/wiki/Scientific_journal" title="Scientific journal">scientific journal</a> article but comments can be made by logged-in users. The comments are also moderated by scientific editors. The intention is to collect practical experience of how to conduct useful <a href="/wiki/Chemical_synthesis" title="Chemical synthesis">chemical synthesis</a> in the lab.  While experimental methods published in an ordinary <a href="/wiki/Academic_journal" title="Academic journal">academic journal</a> are listed formally and concisely, the procedures in ChemSpider SyntheticPages are given with more practical detail. Informality is encouraged. Comments by submitters are included as well. Other publications with comparable amounts of detail include <i><a href="/wiki/Organic_Syntheses" title="Organic Syntheses">Organic Syntheses</a></i> and <i><a href="/wiki/Inorganic_Syntheses" title="Inorganic Syntheses">Inorganic Syntheses</a></i>.  The SyntheticPages site was originally set up by Professors Kevin Booker-Milburn (<a href="/wiki/University_of_Bristol" title="University of Bristol">University of Bristol</a>), Stephen Caddick (<a href="/wiki/University_College_London" title="University College London">University College London</a>), Peter Scott (<a href="/wiki/University_of_Warwick" title="University of Warwick">University of Warwick</a>) and Dr Max Hammond. In February 2010 a merger was announced<sup class="reference" id="cite_ref-23"><a href="#cite_note-23">[23]</a></sup> with the Royal Society of Chemistry's chemical structure search engine ChemSpider and the formation of ChemSpider|SyntheticPages (CS|SP).
</p>
<h3><span class="mw-headline" id="Open_PHACTS">Open PHACTS</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=ChemSpider&amp;action=edit&amp;section=8" title="Edit section: Open PHACTS">edit</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>ChemSpider is serving as the chemical compound repository as part of the <a href="/wiki/OpenPHACTS" title="OpenPHACTS">Open PHACTS</a> project, an <a href="/wiki/Innovative_Medicines_Initiative" title="Innovative Medicines Initiative">Innovative Medicines Initiative</a>. Open PHACTS will deploy an open standards, open access, <a class="mw-redirect" href="/wiki/Semantic_web" title="Semantic web">semantic web</a> approach to address bottlenecks in small molecule drug discovery - disparate information sources, lack of standards and information overload.<sup class="reference" id="cite_ref-24"><a href="#cite_note-24">[24]</a></sup>
</p>
